A good starting point for people who just started using a bunch of css styles

Responsive

Aizio is almost responsive you still need to write your own custom styles only a part of it is responsive and we try to keep it that way. I consider that every designer or developer should take care itself for somethings... but others leave it for Aizio.

Easy To Use

The markup of the code is very easy to use and i try to keep it that way. Most of the plugins that come with Aizio aren't in the official build since i don't want to give you something that you won't use it, instead i will give you the option to choose what you need to use in your projects.

Use What YOU Need

Everything you need you can find here. If not i will build it for you, or you can build it for us and share it with everyone. There will be a section where anyone can upload a certain style that can be used with Aizio so everyone can say thanks and love the guy who made or girl :-)

Plugins

Aizio comes with few things build in the rest you can check out in plugins section and from there start building something custom. I don't want to offer you a bloated css file with things that you might not use at all.

Templates

It comes with a collection of basic layouts that you can style the way you want, but in the end it offers you the ground ups for building great layouts very fast, or showing your client few models of data structure.
